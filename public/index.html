📚 API Bibliothèque Municipale

Node.js · Express · SQLite (Turso / libSQL) · déployé sur Netlify

L'API est disponible sous /api :

  • GET /api — état du service
  • GET /api/livres — liste des livres (?categorie=Roman pour filtrer)
  • GET /api/lecteurs — liste des lecteurs
  • GET /api/emprunts — liste des emprunts

Voir le README.md pour la liste complète des endpoints.
